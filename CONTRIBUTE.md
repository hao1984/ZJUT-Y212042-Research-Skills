# Contribution Notes

本提交按照课程要求通过 Fork + Pull Request 的方式提交。

建议提交步骤：

1. Fork 课程仓库。
2. 将 `skills/paper-grounded-vis-analytics-skill/` 复制到自己 Fork 的 `skills/` 目录下。
3. 提交 commit，并 push 到自己的 Fork。
4. 在课程仓库发起 Pull Request。
5. 根据老师或助教 review 意见继续修改。

提交时无需上传本地完整论文库、向量索引、大体积实验缓存或私有 API key。示例数据与示例输出已放在 `examples/` 中，足以说明 skill 的输入、输出与验证方式。
